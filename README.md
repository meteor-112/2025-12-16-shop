# Shop
一個使用 Bootstrap 與 javascript 製作的商品首頁與購物車
## Screenshot
![preview](./homepage.png)
![preview](./cart-1.png)
![preview](./cart-2.png)

## Features
- 點擊購買商品後跳出成功提示後，自動加入購物車
- 進入購物車顯示購買商品(未購買會提示並返品首頁)
- 購物車自動計算金額
- 填寫資料並記錄顯示

## Tech Stack
- Bootstrap
- SweetAlert
- Swiper

```bash
npm install
